Overview

TP RSI Pro combines three tools into a single pane — an EMA Ribbon, RSI, and Bollinger Bands. Each component serves a distinct purpose and feeds into the signal system.

Trend

EMA Ribbon

10 EMAs forming 5 color-coded bands. Shows trend direction, strength, and potential reversals at a glance.

Momentum

RSI

Color-coded RSI based on position relative to the sensitivity bands. Blue for bullish, red for bearish, white for neutral.

Volatility

Bollinger Bands

Applied directly to RSI for dynamic momentum boundaries. Narrow bands signal low volatility and potential breakouts. Wide bands indicate extended moves.

How it fits with the BST system

TP RSI Pro works as a standalone indicator or alongside Quantum Ribbon Pro. The RSI confirmation toggle inside QR Pro confirms the RSI ribbon alignment — TP RSI Pro gives you full control over every parameter and the complete signal system.

How the Components Work

EMA Ribbon — Trend

10 EMAs grouped into 5 pairs. Shows the current trend state and direction. Each pair is color-coded — blue for bullish alignment, red for bearish.

RSI — Momentum

The RSI line changes color based on its position relative to the sensitivity bands (controlled by RSI Sensitivity Period and Multiplier). When RSI is above the upper sensitivity band it turns blue, below the lower band it turns red, and in between it stays white/neutral.

A gradient fill between the RSI line and the ribbon edge reinforces the momentum direction visually.

Bollinger Bands — Volatility

Applied directly to RSI for dynamic momentum boundaries. Narrow bands signal low volatility and potential breakouts. Wide bands indicate extended moves.

Signal System — BB Breakout + Ribbon Rejection

The primary signal system uses a two-step confirmation process — a BB breakout followed by a ribbon rejection.

BUY Signal

1

RSI breaks above the upper Bollinger Band — setup phase begins. A small ▲ label marks the breakout for situational awareness.

2

RSI pulls back into the ribbon zone.

3

RSI crosses back above the fastest ribbon EMA while in bull color and the required number of ribbon pairs are aligned bullish.

✓

BUY label fires. Setup resets.

SELL Signal

Exact mirror of the BUY condition — RSI breaks below the lower BB, pulls back to ribbon, gets rejected downward with ribbon fully bearish.

Setup Expiry

If the ribbon rejection doesn't occur within the Setup Expiry window (default 50 bars), the setup is cancelled. This prevents stale setups from triggering long after the initial breakout.

Minimum Ribbon Pairs

The Min Ribbon Pairs Aligned setting (1–5) controls how many of the 5 ribbon pairs must be aligned before a signal can fire. Default is 5 (all pairs). Lowering this produces more signals with less confirmation — adjust based on your preferred sensitivity.

Pullback Signals

Pullback Buy and Pullback Sell signals use the same ribbon rejection logic as the primary signals, but without requiring a prior BB breakout. They catch trend continuation entries during established moves when RSI hasn't recently broken out of the bands.

  • ✓ PB Buy — ribbon fully bullish, RSI pulls back into ribbon and crosses above fastest EMA in bull color, RSI within or just reclaiming the bands
  • ✓ PB Sell — mirror condition for bearish ribbon state
  • ✓ Displayed with lighter label color to distinguish from primary signals
  • ✓ Can be toggled independently from primary signals

RSI Volume Profile

Shows which RSI levels have accumulated the most volume over the lookback period. Displayed as horizontal bars alongside the RSI pane.

Reading the Profile

  • Wide bars — high-volume RSI zones.
  • Thin bars — low-volume RSI zones.
  • Point of Control (POC) — the highest-volume RSI level, highlighted in a distinct color.

Adjustable parameters

Lookback period (50–500 bars), number of bins (5–30), bar width, left or right pane placement, and fully customizable colors including the POC highlight.

Multi-Timeframe RSI

Plot a higher timeframe RSI directly on the indicator pane. Enable via MTF RSI Settings. Set your desired timeframe, adjust transparency and line width.

Divergence Detection

Automatically detects regular and hidden divergence between price pivots and RSI pivots. Connector lines between pivot points make the divergence pattern visually clear.

Divergence Types

  • ✓ Regular Bull — price makes lower low, RSI makes higher low. Potential reversal signal.
  • ✓ Regular Bear — price makes higher high, RSI makes lower high. Potential reversal signal.
  • ✓ Hidden Bull — price makes higher low, RSI makes lower low. Trend continuation signal.
  • ✓ Hidden Bear — price makes lower high, RSI makes higher high. Trend continuation signal.

Lines and labels can be toggled independently. Four separate color controls — one per divergence type. Pivot lookback and max lookback range are both adjustable.

Chart Candle Colors

Color your chart candles directly from the RSI pane without adding a separate indicator. Three color modes available:

  • ✓ RSI vs Slowest EMA — candles colored based on whether RSI is above or below the slowest ribbon EMA
  • ✓ RSI vs 50 — candles colored based on RSI above or below the 50 midline
  • ✓ RSI Color — candles match the current RSI color (bull/bear/neutral)

Bull and bear candle colors are fully customizable with adjustable transparency.

Signal History Table

A live log of recent signals displayed on the chart. Shows signal type and bars elapsed since each signal fired.

  • ✓ Persistent ribbon status row — shows current alignment state and pairs count
  • ✓ Configurable number of rows (1–10)
  • ✓ BB Breakout entries can be included or excluded
  • ✓ Position — any of 9 screen locations
  • ✓ Text size — Tiny, Small, Normal
  • ✓ Background transparency adjustable

Alerts & Webhook

Complete alert coverage for every signal type. BUY, SELL, Pullback Buy/Sell, and BB Breakouts also fire JSON alerts for webhook and bot integration.

Settings Reference

Signal Settings

  • ✓ Show Signal Labels — toggle BUY/SELL labels
  • ✓ Show Pullback Signal Labels — toggle PB Buy/Sell labels
  • ✓ Min Ribbon Pairs Aligned — 1–5, how many pairs must align before signal fires
  • ✓ Setup Expiry — 10–300 bars, how long a BB breakout setup remains active
  • ✓ Buy/Sell Label Colors — fully customizable

RSI Settings

  • ✓ RSI Period — default 50
  • ✓ RSI Sensitivity Period — controls neutral zone width. Higher = less sensitive.
  • ✓ RSI Sensitivity Multiplier — scales the neutral zone. Increase to make bull/bear color harder to trigger.
  • ✓ RSI Gradient Fill — toggle fill between RSI and ribbon edge

Ribbon Settings

  • ✓ MA Type — EMA, SMA, WMA, HMA, RMA, ALMA
  • ✓ 10 individual MA lengths — 5 short/long pairs
  • ✓ Ribbon Color Style — Gradient Fade or Classic
  • ✓ Cloud Transparency — controls ribbon fill opacity

Bollinger Band Settings

  • ✓ BB Period — default 50
  • ✓ BB Multiplier — default 2.0. Increase to widen bands, decrease to narrow.
  • ✓ Upper/Lower BB Colors — independent color controls
  • ✓ BB Fill — toggleable with adjustable transparency
